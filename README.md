Réalisation complète d’un pipeline d’analyse des feux de forêt aux États‑Unis : 
nettoyage et traitement du jeu de données d’1,8M de lignes avec Python via Google Colab, 
enrichissement par l’API Open‑Meteo et sources externes, 
puis création d’un reporting interactif Power BI pour le suivi des indicateurs 
et l’analyse des tendances.
